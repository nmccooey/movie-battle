A web app that allows a user to compare movie stats.

Uses Axios and Bulma. Movie data comes from the OMDb API.

A user can search one movie with the left search box and search another movie in the right search box. After both movies are on the screen, the user will be able to compare stats. The winning stats will turn green and the movie with the most winning stats will be the overall winner. The overall winner will be indicated by a green dotted border around the movie's information.
